## Installation

Required: Node 12 or higher

```
$ yarn install
```

## Running the solutions

Run all days

```
yarn start
```

Run specific days (XX in 00...24)

```
yarn start -t "Day XX"
```

Run with optional (slow) tests

```
SLOW=true yarn start
```
